# 🏥 Healthcare Management System Database

## Project Overview

This project is a Healthcare Management System developed using MySQL. It demonstrates the design and implementation of a relational database for managing hospital operations, including patients, doctors, departments, appointments, medical records, laboratory tests, billing, and room allocation.

The database is designed using Entity-Relationship (ER) modeling and normalized up to Third Normal Form (3NF) to ensure data consistency and reduce redundancy.

---

## Features

- Patient Management
- Doctor Management
- Department Management
- Appointment Scheduling
- Medical Records
- Laboratory Test Management
- Billing System
- Room Allocation
- Database Normalization (1NF, 2NF, 3NF)
